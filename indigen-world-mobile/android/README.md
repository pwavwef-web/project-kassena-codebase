# android/ — generated platform scaffolding (pending)

This folder is intentionally a placeholder. The Android native project
(Gradle, manifest, `MainActivity`, etc.) is **generated by the Flutter
toolchain**, not hand-authored, so it was not created during the migration
(the migration environment had Dart but no Flutter SDK).

To populate it, from `indigen-world-mobile/`:

```bash
flutter create . --platforms=android,ios
```

`flutter create` is non-destructive to existing `lib/`, `test/`, `pubspec.yaml`
and `README.md` — it only fills in the missing platform folders and tooling.
